# P2~P5 任务拆解

## P2 扩展业务

### 后端
- 仓库模块
  - 实体/Mapper/Service/Controller：物资、库存、入库、出库、盘点
  - 库存变更原子性：入库/出库/盘点写入与库存更新放在事务内
  - 预警查询：库存量 <= 预警阈值
- 药物模块
  - 实体/Mapper/Service/Controller：药品、批次、发药单/记录
  - 批次扣减策略：按有效期优先（FEFO）扣减 remaining
  - 近效期预警：expiry_date <= today + warningDays
- 打卡模块
  - 实体/Mapper/Service/Controller：打卡记录
  - 支持补录与查询
- 上门服务模块
  - 实体/Mapper/Service/Controller：预约单、服务记录
  - 状态流转约束

### 前端
- 仓库页面：物资档案、入库、出库、库存、盘点
- 药物页面：药品档案、发药管理
- 打卡页面：打卡记录
- 上门服务页面：预约管理、服务记录
- 通知提醒扩展：
  - 药品近效期、库存预警、合同到期提醒（按文档表格）
